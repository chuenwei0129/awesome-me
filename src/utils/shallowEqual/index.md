### shallowEqual

浅比较两个值是否相等，对于对象类型值，仅比较其第一层属性是否相等。

### 参数

| 参数名 | 类型    | 是否必填 | 默认值 | 说明             |
| ------ | ------- | -------- | ------ | ---------------- |
| a      | unknown | 是       | -      | 要比较的第一个值 |
| b      | unknown | 是       | -      | 要比较的第二个值 |

#### 返回值

类型：boolean
如果两个值浅比较相等则返回 true，否则返回 false。
