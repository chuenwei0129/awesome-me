System Setup

Initialize the storage client with your organization credentials. Ensure your environment variables are set securely before deployment.

TypeScript
Python
HTTP
[COPY] 
import { GitStorage } from '@pierre/storage';

const store = new GitStorage({
  name: 'your-org',
  key: env.privateKey,
});

const repo = await store.createRepo({ id: 'new-workspace' });

const result = await repo
  .createCommit({
    targetBranch: 'main',
    message: 'Initial commit',
    author: { name: 'Pierre', email: 'pierre@co.com' }
  })
  .addFileFromString('README.md', '# Getting Started\n')
  .send();

console.log(result.commitSha);

Authentication uses RS256 JWT tokens signed with your private key. Refer to section AUTH_01 for key generation protocols.
