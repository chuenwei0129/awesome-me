# setTimeout

> setTimeout：触发一次后，就停止了，无需手动 clear。当然你可以在它触发前 clear 它，这样它一次都不会触发了。所以，需要还是不需要，不是绝对的，要看你的使用场景。如果你在某种情况下，在计时器触发前需要终止它，那就需要，否则就不需要。
