R — Resources 资源：主题资料库，可能以后用得上，但不承担“责任”
例：代码片段收集、优秀文章、工具用法、读书摘录、各种最佳实践链接

Resources 还能放什么（素材/可复用）

Playbook（操作手册）：发布流程、回滚流程、排查白屏流程、排查接口慢流程

Snippet + 配方：常用 hooks、请求封装、错误边界、表单方案、性能分析脚本

模板库：复盘模板、周报模板、PR 模板、技术方案模板、ADR 模板

工具速查：webpack/vite 配置片段、tsconfig 推荐、eslint/rules 说明

链接 & 摘要：每条链接配 1 ～ 3 行“为什么值得看/用在什么场景”

术语/概念卡片：如“事件循环”“闭包”“Fiber”“Diff”“Hydration”

会议/分享记录：内部 tech talk、外部大会笔记（带结论和行动项）
