## [为什么 CSS 这么难学？](https://www.zhihu.com/question/66167982)

> 任何编程工具都是为人服务的，他们是轮子，是工具，是为了让人提高生产效率的。一切和这个目的相矛盾的设计都是反人类的。

1. CSS 的属性互不正交，大量的依赖与耦合难以记忆。
2. CSS 的很多规则是贯彻整个体系的，而且都记在规范里了，是有规律的，你应该好好读文档而不是去瞎试。
