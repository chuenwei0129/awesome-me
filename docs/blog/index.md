# 📖 个人博客

Hi，我是 **c6i**，一名前端开发者，热爱折腾、记录和分享。

> ❤️ 知识只有经过整理，才不会被时间悄悄带走。

我一直希望，能为每一个问题找到一个漂亮的答案，**那种让人一看就豁然开朗的答案**。

但现实往往很难做到这一点。就像人们总想用一句话说清生活的意义，其实很难。

**带着这种不确定感继续学习，不要有知识包袱，也不要假装自己很聪明。**

世界本来就抽象又复杂，这没什么大不了。
