# Inbox（收件箱）

这里是所有未归类信息的入口：灵感、链接、问题、待办、截屏、代码片段……**先收集，不纠结**。

## 快速入口

- ✍️ 本月收集：[/inbox/2026-02](/inbox/2026-02)
- ✅ 清空流程（5 分钟）
  1. 扫一遍本月页
  2. 能 2 分钟内解决的立刻做掉
  3. 其余按规则归到 P / A / R 或删掉
- 📌 归类规则
  - 有明确目标/期限/交付物 → **Projects**
  - 长期需要维护的标准/流程 → **Areas**
  - 可复用的资料/配方/片段 → **Resources**
  - 不再活跃但想保留 → **Archive**

## 每周清空清单（Checklist）

- [ ] 2 分钟内能完成的（回复消息、改小 bug、记一个结论） → 立刻做
- [ ] 需要推进的事 → 建 Project 页面（写 Next Actions）
- [ ] 需要长期维护的规范/流程 → 放 Areas（补成可执行规则）
- [ ] 可复用的代码片段/排障路径 → 放 Resources（写成 Recipe/Playbook）
- [ ] 没价值/重复/过时 → 删除
- [ ] 归类后在本月页标记为 ✅ Done（保留痕迹）

## 收集时的格式建议（随手写也能搜）

- 任务：`[TODO] ...`
- 问题：`[Q] ...`
- 想法：`[IDEA] ...`
- 资料：`[LINK] ...`
- 结论：`[NOTE] ...`
