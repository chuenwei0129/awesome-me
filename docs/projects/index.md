P — Projects 项目：有明确目标、时间范围、会结束
例：某次项目复盘、面试冲刺（2026 春招）、某个开源仿写、搭一个组件库等
Projects 还能放什么（有目标/会结束）

技术专项：性能优化一期、重构某模块、迁移 React/Vue 大版本

组件库/工具库迭代：v1 发布、补齐文档、做主题系统

学习冲刺：两周 TypeScript 强化、三天 Canvas 小专题

系列输出：工程化系列 8 篇、Hooks 实战系列

Bug/事故复盘：线上事故 2026-02-xx、白屏问题专项排查

开源贡献：给某库提 PR、写 RFC/issue 跟进
